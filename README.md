# HBM2JavaConverter
Convert Hibernate HBM Files to Annotated Java Classes

## Technology stack
- Java
- Ant
- Hibernate
- Hibernate-tools

## How to configure
- Paste your .hbm files to resource folder.
- Open hibernate.cfg.xml and paste the name file.

## Running
```bash
$ mvn clean install
```

**Result**
```
Java classes will be created at src/main/java
```
